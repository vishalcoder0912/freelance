# CareerVeda 2.0

Full‑stack EdTech & Placement platform scaffold.

The repository includes a Next.js (React) frontend with TailwindCSS, Firebase backend (Auth, Firestore, Cloud Functions), and placeholders for AI integration, LMS, project marketplace, and other modules as defined in the PRD.

## Getting Started
```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Further setup steps (Firebase config, environment variables, etc.) are detailed in the docs folder.
